### Author: 29~

## 使用

### 基础于NodeJs NodePPT

```
安装 [Node.Js](https://nodejs.org/en/)
安装 [git](https://git-scm.com/)
```

> 打开命令行 =>

```
npm install nodeppt -g
git clone git@github.com:zp29/wtc-ppt.git
cd wtc-ppt
nodeppt start
```

> 浏览器打开 127.0.0.1:8080
